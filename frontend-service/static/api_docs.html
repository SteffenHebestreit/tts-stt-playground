🎯 Piper Training Service

Status: Healthy

Export: ONNX for PiperTTS deployment

  • POST /train - Start VITS training pipeline

🎤 Qwen3-TTS Service (Voice Cloning)

Port: 5004

Description: Voice cloning and multilingual TTS using Qwen3-TTS

Status: Healthy

Model: Qwen3-TTS-1.7B | Languages: 13 supported

  • POST /tts - Generate speech with built-in speaker
  • POST /clone - Voice cloning with reference audio
  • POST /clone-with-ref-text - Clone with reference text
  • GET /speakers - List available speakers
  • GET /status - System status and GPU info
  • GET /health - Service health check

Voice Cloning Example:

curl -X POST "http://localhost:5004/clone" \
  -F "file=@reference_voice.wav" \
  -F "text=Hello, this is my cloned voice!" \
  -F "lang=en" \
  --output cloned_speech.wav

⚙️ Voice Selection Parameters

  • Language: Auto-detect or specify (en, de, fr, es, it, nl)
  • Quality: x_low, low, medium, high
  • Gender: male, female, any
  • Speed: 0.5x to 2.0x multiplier
  • Format: WAV, MP3 output

Complete TTS Request Example:

curl -X POST "http://localhost:5000/tts" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Bonjour! Comment allez-vous?",
    "language": "fr",
    "quality": "high",
    "gender": "female",
    "speed": 1.0,
    "output_format": "wav"
  }' \
  --output speech.wav

Audio Analysis Example:

curl -X POST "http://localhost:5000/analyze_audio" \
  -F "audio_file=@sample.wav"

# Returns:
{
  "format": "wav",
  "duration": 15.3,
  "sample_rate": 22050,
  "channels": 1,
  "bitrate": 352800,
  "codec": "pcm_s16le"
}
